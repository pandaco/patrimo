Méthodologie
comment on calcule
  • Source des cours : Yahoo Finance, mis en cache 24 h côté backend.
  • Quantité à chaque date : reconstituée en marchant les transactions BUY/SELL dans l'ordre chronologique jusqu'à la date affichée.
  • Valeur portefeuille = Σ (qty × close du jour). Les jours sans cotation (week-ends, jours fériés) reprennent le dernier close connu.
  • Benchmark : Amundi CW8.PA (MSCI World), rebasé pour démarrer à la même valeur que le portefeuille sur la première date où une position est détenue. L'écart entre les deux courbes = ton alpha.
  • Frais et fiscalité : hors valorisation pour le moment. Les frais de courtage sont déjà inscrits dans les transactions BUY.
