jusqu'à 4 instruments du catalogue.

Sélectionne les instruments à confronter, puis lis les caractéristiques structurelles (TER, réplication, devise, éligibilité PEA, indice…) et l'estimation du coût total de détention sur 5 ans pour un DCA mensuel.

Comment utiliser le comparateur

1. Sélectionne jusqu'à 4 instruments parmi la liste pour les afficher côte à côte avec leurs caractéristiques structurelles. Utilise les filtres pour réduire la liste si elle est trop longue.

2. Compare TER (frais annuels), frais cumulés sur 5 ans, éligibilité PEA, distribution capitalisante vs distribuante. Le tableau en bas regroupe tous les critères pour un balayage rapide.

3. Achète via le bouton + Acheter cet instrument sur chaque carte : ça ouvre la boîte de saisie d'une opération BUY pré-remplie avec l'ISIN. Choisis ton enveloppe (PEA, CTO, …), saisis quantité et prix d'achat, valide.

Catalogue. Il te manque un instrument ? Clique + Ajouter un instrument : saisis son ISIN et son ticker, Patrimo vérifie que Yahoo Finance connaît le cours avant de l'accepter. Le nouvel instrument rejoint le catalogue pour la comparaison — il n'apparaîtra dans ton portefeuille qu'après ta première opération d'achat.
